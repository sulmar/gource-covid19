## Tutorial

1. Download file from **Novel Corona virus - COVID19**
https://github.com/datasets/covid-19

2. Lunch converter 
~~~
Usage: converter {input} {output}
~~~

3. Lunch gource 

~~~ bash
gource covid.log -s 0.2
~~~
